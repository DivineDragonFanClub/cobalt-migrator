# cobalt-migrator

This tool can convert non-Cobalt 2.0.0 romfs mods to be Cobalt v0.34.1 compatible.

Get the latest version [here](https://github.com/DivineDragonFanClub/cobalt-migrator/releases).

# How to use
The mod folder you want to convert should have a `romfs` in it. For example, if your mod is `My Awesome Mod`, and it has the `romfs` in it like `My Awesome Mod\romfs`, drag the `My Awesome Mod folder` onto it.

How to use for Windows:
Drag your mod folder onto the .exe.

How to use for Mac:
Using terminal, run ./cobalt-migrator `<path to mod folder with romfs in it>`
